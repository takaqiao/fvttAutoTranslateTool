# PF2e 翻译工具 软件指南

## 1. 功能概览
- 图形界面配置所有参数（带悬停说明）。
- 一键保存配置到 translator_config.json。
- 一键启动翻译脚本。
- 翻译完成后生成差异报告（路径 + A→B + 高亮差异）。
- 支持简体中文后处理（OpenCC）。

## 2. 安装依赖
在项目根目录执行：
- pip install -r requirement.txt

如需强制简体后处理，请确保安装：
- opencc-python-reimplemented

## 3. 使用图形界面
1. 启动图形界面：
   - python gui_app.py
2. 修改参数后点击“保存配置”。
3. 点击“运行翻译”即可启动翻译脚本。

配置会保存到：translator_config.json

## 4. 差异报告
翻译结束后默认生成：translation_diff_report.html
- 格式：路径 + 原值 A + 新值 B + 高亮差异
- 可在配置里关闭或改路径

## 5. 打包为 EXE（Windows）
建议使用 PyInstaller：
1. 安装：
   - pip install pyinstaller
2. 生成 EXE：
   - pyinstaller --onefile --noconsole gui_app.py
3. 生成文件位于：dist/gui_app.exe

## 6. 常见问题
- 未生成简体：请确认已安装 opencc-python-reimplemented 并启用 FORCE_SIMPLIFIED_POST。
- 差异报告为空：表示本次未产生文本变化。
- API Key 优先级：配置文件优先，若为空则使用环境变量。
